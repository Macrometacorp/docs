The data store that Macrometa provides is compatible with almost all Redis API use cases.

:::note
This is a beta feature.
:::

## Capabilities

Because of its fast performance, Redis is a popular choice for:

- caching
- session management
- gaming
- leaderboards
- real-time analytics
- geospatial apps
- ride-hailing apps
- chat/messaging apps
- media streaming apps
- pub/sub apps
